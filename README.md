# ps2

## Features

* Database of PS2 game product codes and artwork.
* Find PS2 game code from ISO.
* Backup PS2 games.
* Rename PS2 ISO files to the OPL standard.

## Install

### Bundler: `gem 'ps2'`

### RubyGems: `gem install ps2`
